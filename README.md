# Computational Physics - 4th Year

This repository contains a collection of practical assignments and simulations for the **Computational Physics** course. The projects focus on implementing numerical methods to solve physical problems, ranging from differential equations to statistical mechanics.

### 1. `EDOs_And_EDPs/`
Numerical solutions for Ordinary (ODEs) and Partial Differential Equations (PDEs).
* **PDEs:** Solvers for **Heat**, **Advection**, **Burgers'**, **Poisson**, **Laplace**, and time-dependent **SchrÃ¶dinger** equations.
* **ODEs:** Analysis of the **Damped Harmonic Oscillator** and **Planetary Motion** using methods like RK4, Verlet, Leapfrog, and Euler.

### 2. `Lattice_And_Molecular_Dynamics/`
Simulations related to solid-state physics and particle dynamics.
* **Crystal Lattices:** Generation of 3D structures (SC, BCC, FCC, Diamond).
* **Molecular Dynamics:** Time evolution of particle systems using **Lennard-Jones potentials** and **Verlet integration** to study forces and temperature.

### 3. `Montecarlo/`
Implementation of Monte Carlo methods for stochastic problems.
* **Integration:** Numerical integration using rejection sampling and calculating hyper-volumes of D-dimensional spheres.
* **Simulation:** Statistical simulations using the **Metropolis Algorithm** .
